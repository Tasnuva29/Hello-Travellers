# Hello-Travellers
> A forum for travellers and alike using ASP.NET Web framework

# Features
> Post (Upvote/Downvote, Comment, Report)\
> Notification\
> Chat\
> Search\
> User Authentication\
> Email Verification\
> User Profile\
> Admin System
